Confidentialité

Des repères simples, sans jargon inutile.

Confidentialité et cadre adulte.

Ce site est un site personnel, adulte et non commercial. Il existe pour permettre un premier message discret, écrit, prudent, sans exposition publique.

Ce que le site ne fait pas

  • aucune inscription ;
  • aucun compte créé ;
  • aucune photo demandée ;
  • aucun téléphone demandé ;
  • aucune prestation vendue ;
  • aucune rencontre promise ;
  • aucun cookie ;
  • aucun outil analytics ;
  • aucun suivi publicitaire ;
  • aucun formulaire serveur ;
  • aucune donnée stockée sur le site.

Le formulaire de contact

Le formulaire de contact est un seuil obligatoire. Certains champs sont nécessaires pour vérifier que le cadre est adulte, clair, respectueux et compatible avec une réponse éventuelle.

Les réponses sont saisies volontairement. Elles peuvent concerner votre âge, votre genre, votre situation personnelle, votre intimité ou votre manière de ressentir les choses. Le site ne les stocke pas : le message est préparé dans votre propre messagerie, et vous pouvez le modifier, le supprimer ou ne pas l’envoyer.

L’adresse e-mail n’est pas affichée publiquement afin d’éviter les messages spontanés ou intempestifs. Le passage par le formulaire protège les deux personnes.

Utilisation des messages

Les informations transmises volontairement servent uniquement à comprendre votre message et décider si une réponse est possible. Un message peut rester sans réponse. L’absence de réponse ne vaut pas jugement : elle signifie simplement que le cadre ne semble pas juste.

Les messages manifestement hors cadre, insistants, agressifs, non adultes ou incompatibles peuvent être ignorés.

Suppression d’un échange

Toute personne peut demander la suppression d’un échange par demande écrite, en repassant par le formulaire de contact si nécessaire.

Cadre adulte

Ce site est réservé aux personnes majeures. Si vous n’êtes pas une personne adulte, vous ne devez pas utiliser le formulaire ni envoyer de message.
